# 目录结构与文件职责

## 目录结构
- src/：核心源码，包含数据、模型、训练、评估与工具模块。
- configs/：配置文件，包含实验、模型、数据与绘图配置。
- scripts/：可执行脚本，包含数据生成、绘图与批量实验入口。
- tests/：测试用例，包含单元测试与最小流程测试。
- docs/：需求、设计与使用文档。
- results/：实验结果输出目录。
- figures/：图表输出目录。

## 文件职责
- src/data/：数据读取、缺失指示m生成、特征处理与批处理。
- src/models/：MAF、TabNet、DM-GFS、SASC、Head、DACOS 及组合模型。
- src/train/：训练入口、评估流程、指标计算、日志与保存。
- src/utils/：通用工具、配置加载、随机种子与日志封装。
- configs/experiments/：主实验与消融实验配置。
- configs/plot.yaml：绘图脚本使用的配置。
- scripts/make_synth_data.py：合成数据生成脚本，含缺失与有序标签。
- scripts/plot_all.py：绘图与报告生成脚本。
- tests/：确保基础模块与冒烟训练可运行。
